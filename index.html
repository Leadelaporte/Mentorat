Projet 2

Voici le projet 2 de la formation Developpeur web d'OpenClassrooms:

Tout d'abord nous allons nous intéresser aux cours. Il y en a 5 :

  1. Mettez en place votre environnment front-end
  2. Apprenez à utiliser la ligne de commande dans un terminal
  3. Gérez du code avec Git et Github
  4. Apprenez à créer votre site web avec html5 et css3
  5. Découpez et intégrez une maquette

Les Livrables

Nous allons maintenant nous intéresser aux livrables. Il y en a 2 :

  1. Un lien vers votre repository Github ou Gitlab
  2. Un lien vers votre page web hébergée en ligne sur Github pages ou sur Gitlab pages, pour présenter votre maquette

Durée

Avec 42 heures en moyennes de cours ainsi que 90 heures de travail sur le projet, il faudra en moyenne 132 heures pour réaliser ce projet.
